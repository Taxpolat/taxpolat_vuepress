# TypeScript

## TS是什么？
::: tip
- `TypeScript`的简称，是一种由微软开发的自由和开源的编程语言。
- TS是JS的超集，简单的说就是在 `JavaScript` 的基础上加入了类型系统
- 相对于JS而言 TS属于强类型语言
:::
### 安装TS
执行命令：
```bash
$ npm install -g typescript 
 //或
$ yarn global add typescript

$ tsc -v // 查看版本
```
